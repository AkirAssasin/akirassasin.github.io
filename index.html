About me

I'm akirassasin, a student. In my free time I make games. A lot of these games are only prototypes but I'll try to release those that are playable.

My main tool is Unity, but occasionally I use Processing or processing.js. I suck a lot at art, so most of my games use programmatic art.

I participate in game jams like Ludum Dare and the Global Game Jam.
